# Интеграция систем координат в Trajectory Planning

## Обзор

Модуль `coordinate_integration` обеспечивает интеграцию систем координат в интерфейс планирования траекторий скважин. Система координат отображается в сайдбаре и применяется ко всем координатам в отчётах, графиках и инклинометрии.

## Функциональность

### Сайдбар выбора CRS

Система координат отображается в сайдбаре Streamlit приложения с:
- Выпадающим списком доступных систем координат
- Переключателем "Автопересчёт координат"
- Отображением текущей выбранной системы

**По умолчанию:** PNO-16 (Зона)

### Доступные системы координат

| Система | Тип | Описание |
|---------|-----|----------|
| PNO-16 (Зона) | Проекция | Нефтегазовая номенклатура, зональная |
| PNO-16 (CM) | Проекция | PNO-16, произвольный меридиан |
| PNO-13 (Зона) | Проекция | Нефтегазовая номенклатура, зональная |
| PNO-13 (CM) | Проекция | PNO-13, произвольный меридиан |
| СК-42 (Градусные) | Географическая | Pulkovo 1942, широта/долгота |
| СК-42 Зона 6-20 | Проекция | Гаусс-Крюгер из Pulkovo 1942 |
| Пулково 1995 | Географическая | Современный датум |
| Пулково 1995 Зона 13, 18 | Проекция | Гаусс-Крюгер из Pulkovo 1995 |
| ГСК-2011 | Географическая | Современная государственная |
| WGS84 | Географическая | Глобальная GPS система |

### Интеграция в результаты

При переключении системы координат:
1. Заголовок секции "Результаты расчёта" обновляется с суффиксом CRS
2. Координаты в таблицах форматируются соответственно типу CRS
3. Инклинометрия отображается в выбранной системе
4. Графики обновляются с соответствующими подписями

## Использование

```python
from pywp.coordinate_integration import (
    render_crs_sidebar,
    get_selected_crs,
    get_crs_display_suffix,
)

# В основном коде страницы
selected_crs = render_crs_sidebar()

# Применение к результатам
display_suffix = get_crs_display_suffix(selected_crs)
st.markdown(f"## Результаты{display_suffix}")
```

## API Reference

### `render_crs_sidebar()`

Рендерит сайдбар с выбором системы координат.

**Returns:** `CoordinateSystem` — выбранная система координат

### `get_selected_crs()`

Получает выбранную систему координат из состояния сессии.

**Returns:** `CoordinateSystem` — текущая система координат (по умолчанию PNO_16_ZONE)

### `get_crs_display_suffix(crs)`

Возвращает короткий суффикс для отображения в заголовках.

**Args:**
- `crs: CoordinateSystem` — система координат

**Returns:** `str` — суффикс для отображения, например " (PNO16)"

### `format_coordinates_for_display(x, y, z, crs)`

Форматирует координаты для отображения в UI.

**Args:**
- `x, y, z: float` — координаты
- `crs: CoordinateSystem` — целевая система координат

**Returns:** `tuple[str, str, str]` — отформатированные координаты

## Зависимости

- `pywp.coordinate_systems` — базовые системы координат
- `streamlit` — UI компоненты
- `pandas` — работа с таблицами инклинометрии

## Примечания

- Автоматический пересчёт координат требует `pyproj` для преобразований между датумами
- Без `pyproj` отображаются исходные координаты с переименованием колонок
- PNO-16/PNO-13 требуют уточнения (зона vs CM) на основе easting > 1,000,000
